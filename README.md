### 💼 Data Industry Salaries Dashboard

### 📌 Project Overview

The Data Industry Salaries Dashboard is an interactive Power BI / Tableau solution supported by Python, SQL, and Excel.
It analyzes salary data across job roles, experience levels, regions, and company sizes to uncover pay trends in the data industry.
This project empowers professionals and employers with data-driven insights for career planning and hiring strategies.

---

### 🚀 Key Features

📊 Interactive Dashboard in Power BI/Tableau to explore salary patterns.

🧹 Data Cleaning & Transformation in Python & Excel for accuracy.

🗄️ SQL-based queries for aggregation and slicing by role, country, and experience.

🌍 Regional comparison of salary ranges across industries.

📈 Trend analysis to highlight compensation growth with seniority.

---
